# RoomList Chrome Web Store 上架文案

## 名稱

RoomList 空間採購清單

## 簡短說明

跨電商收藏家居商品，依空間整理採購清單、數量與預算；資料只儲存在本機。

## 詳細說明

RoomList 幫你把不同電商網站的家居商品集中成一份空間採購清單。

在支援的商品頁選擇「加入採購清單」，即可把商品名稱、價格、圖片與來源連結保存到 Chrome 本機，並依客廳、臥室、書房等空間分類。

主要功能：

- 跨電商收藏家居商品
- 依空間分類並計算小計與總額
- 修改商品數量、價格與空間
- 保留商品來源網站及原始連結
- 匯出 PDF 採購清單
- 透過自己的郵件程式分享清單內容
- 不需註冊或登入 RoomList 帳號

隱私保護：

商品、清單、空間分類及選填的預設收件 Email 只儲存在目前的 Chrome 瀏覽器設定檔，不會傳送到 RoomList 開發者伺服器，也不會用於廣告或販售。

電商網站可能調整商品頁結構，商品價格、折扣與庫存請以原網站結帳頁為準。

RoomList 為獨立開發工具，與支援的電商品牌沒有隸屬、合作、贊助或授權關係。
